# Training Planner

Автор: Таучев Тахир

## Описание
Программа предназначена для планирования тренировок. Позволяет добавлять тренировки с датой, типом и продолжительностью, сохранять их, фильтровать по типу и дате. Все данные сохраняются в JSON-файл.

## Инструкция по запуску
1. Установите Python 3.x
2. Сохраните скрипт в файл `training_planner.py`
3. Запустите командой: `python training_planner.py`

## Использование
- Введите дату в формате ГГГГ-ММ-ДД.
- Введите тип тренировки.
- Введите длительность в минутах.
- Нажмите «Добавить тренировку».
- Используйте фильтры для поиска по типу и дате.
- Для удаления выберите тренировку и нажмите «Удалить выбранное».
- Данные сохраняются автоматически.

## Тестовые сценарии
- Добавление нескольких тренировок.
- Фильтрация по типу и дате.
- Удаление тренировок.
